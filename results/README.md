# Results

`results/report/` is the curated reporting area. Raw model outputs, cached
forecasts, intermediate experiment folders, and large diagnostics stay local
unless they are promoted into this folder.

## Curated Report Artifacts

- `results/report/master_results_table.csv`
- `results/report/master_results_headline.csv`
- `results/report/seasonal/`
- `results/report/transition/`
- `results/report/landsurface/`
- `results/report/regionalization/`
- `results/report/paper/`

`results/report/paper/` is the manuscript-facing evidence pack: final tables,
figures, and short manuscript-drafting notes generated by
`scripts/generate_manuscript_results.py`.

## Local Supporting Results

The following folders can exist locally but are ignored unless promoted into
`results/report/`:

- `results/multiregion/`
- `results/regionalization/`
- `results/seasonal/`
- `results/temporal/`
- `results/validation/`
- `results/landsurface/`
- `results/transition/`

Archived checkpoint folders are historical references only and should not be
used as current conclusions.
