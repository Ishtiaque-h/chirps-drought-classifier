# Literature Folder

This folder stores the related-work index, local PDFs, and protocol notes used
for manuscript planning.

## Files

- `related_works_index.csv` tracks papers, source URLs, local PDF status, and
  how each source affects the project decision.
- `literature_synthesis_memo.md` summarizes the main research implications and
  next-experiment logic.
- `literature_protocol_audit.csv` compares related work by target, lead,
  predictors, validation, metrics, and comparability to this project.
- `literature_protocol_audit.md` summarizes supported by
  the protocol audit.
- `papers/` contains local PDFs where direct download was available.

## Use In Manuscript

The protocol audit is prepared to avoid overclaiming or shortsighting against prior studies. Since many studies are not directly comparable to our strict monthly SPI-1 probability task, it is not that prior studies are broadly invalid. Rather, we are just trying to establish and prove our hypothesis by showing and validating with broader evidence that drought forecasting with spi-1 lead-1 is a hard problem with lots of predictability barrier.
