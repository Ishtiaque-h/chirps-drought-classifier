## 1. Primary Results (Must Share)

These four outputs together constitute the core scientific contribution of the project.

### 1.1 Skill Score Table

| Output file | `outputs/forecast_skill_bss_hss_table.csv` |
|---|---|
| **Generated by** | `scripts/evaluate_forecast_skill.py` |
| **Why significant** | Single most important result. Compares all five ML models plus three naive baselines on Brier Skill Score, HSS, and ROC-AUC, with 95% bootstrap confidence intervals. Directly answers the research question: can ML beat climatology? |

Key numbers (63 test months, 2021–2026):

| Forecaster | Brier Score (dry) | BSS vs. climatology | HSS (3-class) | ROC-AUC (dry) |
|---|---:|---:|---:|---:|
| Climatological baseline | 0.0646 (ref) | 0.00 | 0.00 | — |
| Persistence | 0.1011 | −0.57 | 0.09 | 0.56 |
| SPI-1 heuristic | 0.0949 | −0.47 | 0.09 | 0.56 |
| Logistic Regression | 0.0874 | −0.35 | 0.15 | 0.81 |
| Random Forest | 0.0820 | −0.27 | 0.11 | 0.60 |
| XGBoost | 0.0687 | −0.06 | 0.00 | 0.67 |
| **XGBoost-Spatial** | **0.0666** | **−0.03** | 0.00 | **0.68** |
| ConvLSTM | 0.0823 | −0.27 | 0.22 | 0.52 |

> **Key finding:** No model achieves BSS > 0 (i.e., no model beats climatology in calibrated probability skill). XGBoost-Spatial comes closest (BSS = −0.03) with a 95% CI spanning zero — statistically indistinguishable from climatology. ROC-AUC ≈ 0.68 confirms ranking signal exists, but models cannot translate it into reliable probabilities.

### 1.2 Calibration Study Results

| Output file | `outputs/calib_study_results.csv` |
|---|---|
| **Generated by** | `scripts/evaluate_forecast_skill.py` |
| **Why significant** | Documents Brier Score decomposition (reliability + resolution − uncertainty) and p-values for three calibration methods (uncalibrated, Platt scaling, isotonic regression). Proves that the BSS gap cannot be closed by post-hoc calibration — the bottleneck is resolution (discrimination), not reliability (over-confidence). |

## 6. Reproducibility Reference

All significant outputs above are fully reproducible from the public pipeline:

```bash
# Full pipeline (see README.md for prerequisites)
bash scripts/download_chirps_v3_monthly_1991_2026.sh
python scripts/clip_to_cvalley_monthly.py
python scripts/make_spi_labels.py
python scripts/build_dataset_forecast.py
python scripts/train_forecast_logreg.py
python scripts/train_forecast_rf.py
python scripts/train_forecast_xgboost.py
python scripts/train_forecast_xgb_spatial.py
python scripts/evaluate_forecast_skill.py        # → skill table + calibration outputs
python scripts/xgb_shap_forecast_analysis.py     # → SHAP plots
python scripts/validate_era5_spi.py              # → ERA5 cross-validation
python scripts/validate_usdm.py                  # → USDM consistency
python scripts/plot_spatial_skill.py             # → spatial accuracy map
python scripts/plot_case_study.py                # → 2021–2026 case study
```

- **Environment:** `environment.yml` (conda) or `requirements.txt` (pip)
- **Random seed:** 42 (set in all training scripts)
- **Python version:** 3.10+
- **Full assessment and research roadmap:** [`ANALYSIS.md`](../ANALYSIS.md)
